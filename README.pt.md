# 📸 Selfie Snapper

O Selfie Snapper permite **capturar** selfies incríveis diretamente do seu navegador usando a webcam do seu dispositivo.

![SelfieSnapperWebDemo](https://github.com/KrishBharadwaj5678/SelfieSnapper/raw/main/demos/SelfieSnapperWebDemo.png)
![SelfieSnapperMobileDemo](https://github.com/KrishBharadwaj5678/SelfieSnapper/raw/main/demos/SelfieSnapperMobileDemo.jpeg)

## ✨ Recursos

| Recurso               | Descrição                                          |
| --------------------- | -------------------------------------------------- |
| 📷 Acesso à webcam    | Capture selfies usando a câmera do seu dispositivo |
| ⚡ Captura instantânea | Tire fotos instantaneamente com apenas um clique   |
| 🎨 Interface limpa    | Interface simples e fácil de usar                  |
| 💾 Baixar imagens     | Salve suas selfies capturadas localmente           |
| 🌐 Baseado na web     | Funciona diretamente no navegador                  |
| 📱 Design responsivo  | Funciona em computadores e dispositivos móveis     |

## 🛠️ Tecnologias utilizadas

| Tecnologia       | Finalidade                           |
