# 📸 Selfie Snapper

Selfie Snapper позволяет **делать** потрясающие селфи прямо в браузере с помощью вашей веб-камеры.

![SelfieSnapperWebDemo](https://github.com/KrishBharadwaj5678/SelfieSnapper/raw/main/demos/SelfieSnapperWebDemo.png)
![SelfieSnapperMobileDemo](https://github.com/KrishBharadwaj5678/SelfieSnapper/raw/main/demos/SelfieSnapperMobileDemo.jpeg)

## ✨ Возможности

| Возможность             | Описание                                     |
| ----------------------- | -------------------------------------------- |
| 📷 Доступ к веб-камере  | Создание селфи с помощью камеры устройства   |
| ⚡ Мгновенная съёмка     | Делайте фотографии одним нажатием кнопки     |
| 🎨 Чистый интерфейс     | Простой и удобный пользовательский интерфейс |
| 💾 Загрузка изображений | Сохраняйте сделанные селфи локально          |
| 🌐 Веб-приложение       | Работает прямо в вашем браузере              |
| 📱 Адаптивный дизайн    | Поддержка компьютеров и мобильных устройств  |

---

## 🛠️ Используемые технологии

| Технология       | Назначение                                   |
| ---------------- | -------------------------------------------- |
| 🌐 **HTML5**     | Структура веб-приложения                     |
| 🎨 **CSS3**      | Стилизация и адаптивный дизайн               |
| ⚡ **JavaScript** | Управление доступом к веб-камере и функциями |

## 🚀 Начало работы

### 1️⃣ Клонирование репозитория

```bash
git clone https://github.com/KrishBharadwaj5678/SelfieSnapper.git
```

### 2️⃣ Перейдите в папку проекта

```bash
cd SelfieSnapper
```

### 3️⃣ Откройте в браузере

Просто откройте файл `index.html` в вашем браузере.
