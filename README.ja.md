# 📸 Selfie Snapper

Selfie Snapper は、ウェブカメラを使用してブラウザ上から直接**美しいセルフィーを撮影**できるアプリケーションです。

![SelfieSnapperWebDemo](https://github.com/KrishBharadwaj5678/SelfieSnapper/raw/main/demos/SelfieSnapperWebDemo.png)
![SelfieSnapperMobileDemo](https://github.com/KrishBharadwaj5678/SelfieSnapper/raw/main/demos/SelfieSnapperMobileDemo.jpeg)

## ✨ 機能

| 機能            | 説明                    |
| ------------- | --------------------- |
| 📷 ウェブカメラアクセス | デバイスのカメラを使用してセルフィーを撮影 |
| ⚡ 即時キャプチャ     | ワンクリックですぐに写真を撮影       |
| 🎨 シンプルなUI    | 使いやすく直感的なインターフェース     |
| 💾 画像のダウンロード  | 撮影したセルフィーをローカルに保存     |
| 🌐 ウェブベース     | ブラウザ上で直接動作            |
| 📱 レスポンシブデザイン | PCやモバイルデバイスに対応        |

---

## 🛠️ 技術スタック

| 技術               | 目的                |
| ---------------- | ----------------- |
| 🌐 **HTML5**     | ウェブアプリケーションの構造を作成 |
| 🎨 **CSS3**      | スタイリングとレスポンシブデザイン |
| ⚡ **JavaScript** | ウェブカメラ制御とアプリ機能の実装 |

---

## 🚀 はじめに

### 1️⃣ リポジトリをクローンする

```bash
git clone https://github.com/KrishBharadwaj5678/SelfieSnapper.git
```

### 2️⃣ プロジェクトフォルダへ移動する

```bash
cd SelfieSnapper
```

### 3️⃣ ブラウザで開く

`index.html` ファイルをブラウザで開くだけで使用できます。
